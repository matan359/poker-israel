# 🔐 גישה לממשק האדמין

## איך להתחבר לממשק האדמין?

ממשק האדמין זמין רק למשתמשים עם הרשאות אדמין. יש שתי דרכים לקבל גישה:

### שיטה 1: התחברות עם אימייל אדמין

1. **התחבר עם האימייל**: `admin@pokerisrael.com`
2. **הכנס ללובי** - הכפתור "🎛️ ADMIN" יופיע אוטומטית בתפריט העליון
3. **לחץ על הכפתור** כדי לפתוח את ממשק האדמין

### שיטה 2: עדכון משתמש קיים לאדמין

אם אתה כבר משתמש רשום ורוצה להפוך לאדמין:

1. **פתח את Firebase Console**
2. **עבור ל-Firestore Database**
3. **מצא את הקולקציה `users`**
4. **מצא את המסמך של המשתמש שלך** (לפי ה-UID או האימייל)
5. **עדכן את המסמך** והוסף את השדה:
   ```json
   {
     "isAdmin": true
   }
   ```
6. **רענן את הדף** - הכפתור "🎛️ ADMIN" יופיע

### מה אפשר לעשות בממשק האדמין?

#### 📊 שולחנות
- **צור שולחן חדש** - הוסף שולחנות פוקר חדשים
- **ערוך שולחנות** - שנה הגדרות של שולחנות קיימים
- **מחק שולחנות** - הסר שולחנות מהמערכת

#### 💰 רווחים
- **צפה בסיכום רווחים** - סך כל הרווחים של הבית
- **היסטוריית רווחים** - כל העסקאות (עמלות דילר, טיפים)

#### ⚙️ הגדרות
- **עמלת דילר** - הגדר את אחוז העמלה
- **עמלת בית** - הגדר את אחוז העמלה של הבית
- **צ'יפים התחלתיים** - הגדר כמה צ'יפים מקבל שחקן חדש

### הערות חשובות

- **אבטחה**: רק משתמשים עם `isAdmin: true` או `email: 'admin@pokerisrael.com'` יכולים לגשת לממשק
- **Firestore Rules**: ודא שהכללים ב-Firestore מאפשרים למשתמשים עם הרשאות אדמין לערוך את הנתונים
- **גיבוי**: מומלץ לגבות נתונים לפני ביצוע שינויים משמעותיים

### בעיות נפוצות

**הכפתור לא מופיע?**
- ודא שהמשתמש שלך מסומן כ-`isAdmin: true` ב-Firestore
- או התחבר עם `admin@pokerisrael.com`
- רענן את הדף

**אין הרשאות לערוך?**
- בדוק את כללי האבטחה ב-Firestore
- ודא שהמשתמש שלך מוגדר נכון
