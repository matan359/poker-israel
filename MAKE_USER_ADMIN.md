# 🔐 איך להפוך משתמש לאדמין ב-Firebase Console

## שיטה 1: דרך Firebase Console (הכי פשוט)

### שלבים:

1. **פתח Firebase Console**
   - לך ל: https://console.firebase.google.com/
   - בחר את הפרויקט: **poker-3c5c6** (או הפרויקט שלך)

2. **עבור ל-Firestore Database**
   - בתפריט השמאלי, לחץ על **Firestore Database**
   - לחץ על הטאב **Data** (אם לא כבר שם)

3. **מצא את הקולקציה `users`**
   - בחלק השמאלי, תמצא את הקולקציה **users**
   - לחץ עליה

4. **מצא את המשתמש שלך**
   - תראה רשימה של כל המשתמשים
   - כל מסמך הוא משתמש (ה-ID הוא ה-UID של המשתמש)
   - או חפש לפי האימייל בתוך המסמך

5. **ערוך את המסמך**
   - לחץ על המסמך של המשתמש
   - לחץ על **Edit document** (ערוך מסמך)
   - הוסף שדה חדש:
     - **Field name**: `isAdmin`
     - **Field type**: `boolean`
     - **Value**: `true`
   - לחץ **Update**

6. **או דרך JSON Editor**
   - לחץ על המסמך
   - לחץ על **Edit document**
   - לחץ על **JSON** (בחלק העליון)
   - הוסף את השורה: `"isAdmin": true`
   - לחץ **Update**

## שיטה 2: דרך ממשק האדמין (אם יש לך גישה)

אם יש לך כבר משתמש אדמין אחד, תוכל להשתמש בממשק האדמין:

1. התחבר כמנהל
2. פתח את ממשק האדמין (🎛️ ADMIN)
3. עבור לטאב "הגדרות"
4. גלול למטה לסעיף "ניהול משתמשים"
5. לחץ על "הפוך לאדמין" והזן את האימייל

## שיטה 3: דרך Firebase CLI (למתקדמים)

אם יש לך Firebase CLI מותקן:

```bash
firebase firestore:set users/USER_ID isAdmin true
```

## ⚠️ חשוב:

- **אחרי שעדכנת**, רענן את הדף באפליקציה
- הכפתור "🎛️ ADMIN" יופיע אוטומטית
- אם לא מופיע, בדוק ש-`isAdmin: true` נשמר נכון

## איך לבדוק שהצלחת:

1. רענן את הדף
2. בדוק שהכפתור "🎛️ ADMIN" מופיע בתפריט העליון
3. אם לא מופיע, פתח את ה-Console (F12) ובדוק אם יש שגיאות

## אם אין לך משתמשים ב-users collection:

אם המשתמש לא קיים ב-`users` collection:
1. התחבר לאפליקציה עם המשתמש הזה
2. המשתמש יווצר אוטומטית
3. אז תוכל לערוך אותו ולהוסיף `isAdmin: true`
